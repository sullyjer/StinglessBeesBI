# 🐝 Stingless Bee Species Analysis – Brazil

---

## 📊 Project Overview
This project was developed to structure, analyze, and visualize information about stingless bee species in Brazil. The main objective is to transform unstructured data into actionable insights through data engineering and visualization techniques.

The final deliverable is an interactive **Power BI dashboard**, built after a full pipeline of data collection, normalization, and enrichment.

---

## 🔍 Data Collection
The dataset was created through a **web crawling process**, extracting publicly available information about stingless bee species.

**Collected data includes:**
- Species names  
- Taxonomic classification  
- Geographic distribution  
- Additional descriptive attributes  

---

## 🧹 Data Processing & Normalization
After extraction, the raw data underwent a structured transformation process:

- ✔️ Data cleaning (removal of inconsistencies and duplicates)  
- ✔️ Standardization of fields and formats  
- ✔️ Normalization into relational structures  
- ✔️ Data enrichment with additional attributes  

Special attention was given to ensure data consistency and usability for analytical purposes.

---

## 🔗 Data Enrichment
A complementary dataset was integrated to enhance the analysis:

- 🔗 External card links for species (when available)  
- 🔄 LEFT JOIN to preserve all base records  
- ⚪ Null handling for non-listed species  

This ensures a complete dataset while maintaining referential integrity.

---

## 📈 Power BI Dashboard
A **Power BI report** was built on top of the processed dataset, enabling:

- 📊 Interactive exploration of species data  
- 🎯 Filtering and segmentation  
- 🌎 Insights into distribution and classification  
- 🔍 Easy navigation through enriched information  

---
